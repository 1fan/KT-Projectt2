There are two java files, one is main.java, one is FileOperation.java.

In the main.java, main function calls the FileOperation class instance and calls the ReadTXTWriteARFF() function.
In the FileOperation.java, there are three Arraylist to store the attributes read from the attribute folder.These three
Arraylists are used to judge the attributes for each tweet. Then in the ReadTXTWriteARFF() function, it conducts the
file operations, for instance read file, write file, as well as the attribute check functions. The output file is written
in the given path.
